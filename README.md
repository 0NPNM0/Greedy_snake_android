安卓贪吃蛇游戏
有四个关卡选择，分别是中国内地版，中国澳门版，中国香港版和中国台湾版。
不同的版本代表贪吃蛇不同的前进速度。
本项目中除了启动页的插图为个人绘制，其余插图均由stable diffusion生成。
